
1. wget url_of_file: Download a file from a URL.
2. wget -O <name> url_of_file: Download a file and save it with a specific name.
3. curl <api-call>: Make an API call or request data from a server.
4. ssh user@hostname: Access a remote Linux server via SSH.
5. scp file-name user@hostname:/path/: Securely copy a file to a remote server.
6. telnet IP PORT: Establish a telnet connection.
7. traceroute www.google.com: Trace the path to an external website or server.
